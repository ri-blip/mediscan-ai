    // pipeline_architecture
    How It Works.
    →
    01 — preprocess
    MRI Enhancement
    CLAHE, gamma correction, histogram equalization, noise removal & Canny edge detection
    OpenCV · NumPy · scikit-image
    →
    02 — encode
    Vision Encoder
    Pre-trained ViT extracts spatial feature embeddings from the enhanced MRI scan
    PyTorch · Hugging Face · ViT
    →
    03 — retrieve
    RAG Layer
    FAISS vector search retrieves top-3 grounding docs from medical literature corpus
    LangChain · FAISS · SentenceTransformers
    →
    04 — generate
    VLM Report
    LLaVA-Med generates Findings, Impression & Recommendation grounded in retrieved context
    LLaVA · Ollama · LangChain
    05 — serve
    API + Deploy
    FastAPI endpoint, Docker containerized, deployed on AWS EC2 with Gradio demo UI
    FastAPI · Docker · AWS EC2 · Gradio
